# Comands

### ng g m [name-module]

- Create a module

### ng g c [path/name-component]

- Create a component

### ng g c [path/name-component] --skip-tests -s

- Create a component

\*\* Do not create "spec.ts" test files for the application.

```
--skip-tests
```

\*\* Include styles inline in the root component.ts file

```
-s
```
